# Installation

## Stable release

To install Subscribly Churn Prediction, run this command in your terminal:

```sh
uv add Subscribly
```

Or if you prefer to use `pip`:

```sh
pip install Subscribly
```

## From source

The source files for Subscribly Churn Prediction can be downloaded from the [Github repo](https://github.com/AvanindraBose/Subscribly).

You can either clone the public repository:

```sh
git clone git://github.com/AvanindraBose/Subscribly
```

Or download the [tarball](https://github.com/AvanindraBose/Subscribly/tarball/master):

```sh
curl -OJL https://github.com/AvanindraBose/Subscribly/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd Subscribly
uv pip install .
```
